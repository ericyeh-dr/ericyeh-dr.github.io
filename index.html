Ophthalmology surgical video analysis

Deep learning-based automatic real-time analysis of surgical performance.

Given a surgical video, the model identifies, for every frame in real-time, which step the frame is in, which instruments are being used, and how skillful they are being used.

This serves a good tool for surgical trainee to track their progress and identify which part they need the most practice.

Ophthalmology progress note summary

Automatic summary of ophthalmology progress note using weakly supervised learning

The problem with scarce data has prohibited the progress of language model application on medical notes. In this project I used a weakly supervised learning that utilized an imperfect label to infer the importance of each sentence and build a summary based on the scores.
